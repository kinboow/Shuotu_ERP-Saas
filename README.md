# 跨境电商ERP系统

## 📋 项目简介

跨境电商ERP系统是一个功能完整的企业资源规划系统，专为跨境电商业务设计，支持多平台店铺管理、商品同步、订单处理、库存管理和财务管理等核心功能。

### 核心特点

- 🌐 **多平台支持** - 统一管理SHEIN、Amazon、eBay等多个电商平台
- 🔐 **安全认证** - 双数据库架构，独立的认证系统
- 📊 **数据同步** - 自动同步商品、订单等数据
- 💰 **财务管理** - 完整的资金流水和提现管理
- 📦 **库存管理** - 备货订单和库存跟踪
- 🎨 **用户友好** - 现代化的UI设计，响应式布局

## 🚀 快速开始

### 环境要求
- Node.js 14.0+
- MySQL 5.7+
- npm 6.0+

### 安装步骤

```bash
# 1. 克隆项目
git clone <repository-url>
cd cross-border-erp

# 2. 配置数据库
mysql -u root -p
CREATE DATABASE cross_border_erp CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
mysql -u root -p cross_border_erp < database/schema.sql

# 3. 配置后端
cd backend
npm install
cp .env.example .env
# 编辑 .env 文件，填写数据库连接信息
npm run init-platform
npm start

# 4. 配置前端
cd frontend
npm install
npm start

# 5. 访问系统
# http://localhost:3000
```

## 🔧 配置说明

### 环境变量配置

编辑 `backend/.env` 文件：

```env
# A数据库配置（ERP业务数据库）
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=cross_border_erp

# B数据库配置（认证数据库）
AUTH_DB_HOST=localhost
AUTH_DB_PORT=3306
AUTH_DB_USER=root
AUTH_DB_PASSWORD=your_password
AUTH_DB_NAME=192_168_5_40

# JWT配置
JWT_SECRET=your_random_secret_key
JWT_EXPIRES_IN=7d

# 服务器配置
PORT=5000
```
