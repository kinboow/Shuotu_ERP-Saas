Create and list products intended for sale exclusively in local shops.‚Äã
You can create products in AVAILABLE categories. (US sellers can also create products in INVITE_ONLY categories). After creation, it will be sent for audit review by TikTok Shop. Use the Product status change webhook to keep track of the review status.‚Äã
Note: ‚Äã
- This API is applicable for all sellers.‚Äã
- Global sellers who have migrated to use the local replication listing method can use this API to create products. Otherwise, they can continue to use the Create Global Product API to create global products.‚Äã
- Before calling this API, we recommend that you prepare the necessary information by following the usage flow for your region.‚Äã
- There may be a limit to the number of products you can list per day. We recommend prioritizing the creation of key products first to ensure they get published. Refer to TikTok Shop Academy for details on the limit.‚Äã
- The language used in the product content must align with the target market's language (e.g. don't use Chinese), otherwise the listing will fail or be rejected.‚Äã
