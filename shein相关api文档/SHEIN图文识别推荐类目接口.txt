图文识别推荐类目
POST
/open-api/goods/image-category-suggestion
自运营
全托管
半托管
支持通过商品图片或商品文本描述获取推荐类目，仅返回当前商家可用类目范围内的推荐类目。

可实现3种入参组合：仅图片、仅文本、图片+文本。

请求参数（请求体）
收起
参数名称	数据类型	是否必须	描述
url
string	否	
图片url。可使用外部图片地址，仅支持格式JPG/JPEG/PNG，大小≤3MB。

url、productInfo，必须入参其中一个值，也可两个都入参。

productInfo
string	否	
商品的文案信息。可输入商品描述或商品名称等任意和商品相关的文本信息，不限语种，1000个字符内，不支持emoji。

url、productInfo，必须入参其中一个值，也可两个都入参。
