查询商品备货信息列表
POST
/open-api/openapi-business-backend/stock-goods-list
全托管
通过该接口获取商品备货相关信息；该接口适用全托管模式，部分自运营、半托管模式商家；

string	是	
请求时间戳，格式为时间转换为毫秒的值，也就是从1970年1月1日起至今的时间转换为毫秒，如：1583398764000

string	是	
接口签名
