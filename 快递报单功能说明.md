# 快递报单功能说明

## 📱 功能概述

快递报单功能是为快递员设计的移动端报单工具，可以快速记录每日的快递收件情况，并在电脑端实时查看统计数据。

---

## 🎯 使用场景

### 场景1：快递员每日报单
快递员每天上门收件后，需要向公司报告当天收了多少件快递，分别是哪些包裹号。

**传统方式**：
- 手写记录，容易出错
- 回公司后再录入系统，效率低
- 无法实时统计

**使用本功能**：
- 手机扫码即可记录
- 自动分类大件/小件
- 实时提交到系统
- 管理员立即可见

### 场景2：仓库收发货统计
仓库需要统计每天各快递公司的收发货数量。

**使用本功能**：
- 按快递公司分类统计
- 按日期范围查询
- 导出报表分析

## 📱 移动端操作流程

### 步骤1：打开报单页面

1. 打开PDA应用：`https://192.168.5.105:3099`
2. 登录账号
3. 在首页点击"快递报单"图标

### 步骤2：填写基本信息

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 快递公司 | 从下拉列表选择（顺丰、中通等） | ✅ 必填 |
| 报单日期 | 自动填充今天日期，可修改 | ✅ 必填 |
| 大件数量 | 点击+/-按钮调整 | ⭕ 选填 |
| 小件数量 | 点击+/-按钮调整 | ⭕ 选填 |
| 备注 | 其他说明信息 | ⭕ 选填 |

### 步骤3：扫描包裹号（可选）

如果需要记录具体的包裹号：

1. 在"扫描包裹号"输入框中：
   - 使用扫码枪扫描快递单号
   - 或手动输入包裹号
2. 点击"添加"按钮
3. 选择包裹类型：
   - 点击"大件"：大件数量+1
   - 点击"小件"：小件数量+1
4. 包裹会显示在下方列表中
5. 如需删除，点击包裹右侧的删除按钮

**提示**：
- 扫描后会自动清空输入框，可继续扫描下一个
- 重复的包裹号会提示，不会重复添加
- 每个包裹都会记录扫描时间

### 步骤4：提交报单

1. 确认信息无误
2. 点击底部"提交报单"按钮
3. 提交成功后：
   - 显示成功提示
   - 表单自动清空
   - 可继续填写下一单

## 💻 电脑端查看

### 访问报单管理页面

1. 打开Web管理端：`https://erp.hlsd.work:3000`
2. 登录账号
3. 访问：`/courier-reports` 或在菜单中找到"快递报单"

### 功能说明

#### 1. 统计卡片（顶部）

显示各快递公司的汇总数据：
- 总件数
- 大件数量
- 小件数量

可按日期范围筛选统计数据。

#### 2. 筛选条件

| 筛选项 | 说明 |
|--------|------|
| 快递公司 | 选择特定快递公司 |
| 状态 | 已提交/已确认/已取消 |
| 报单编号 | 模糊搜索 |
| 日期范围 | 选择起止日期 |

#### 3. 报单列表

显示所有报单记录，包含：
- 报单编号（自动生成）
- 快递公司
- 报单日期
- 大件数量（红色显示）
- 小件数量（蓝色显示）
- 总件数（粗体显示）
- 操作员姓名
- 状态标签
- 创建时间

#### 4. 操作按钮

| 操作 | 说明 | 权限 |
|------|------|------|
| 详情 | 查看报单详细信息和包裹列表 | 所有人 |
| 确认 | 将状态改为"已确认" | 仅"已提交"状态 |
| 取消 | 将状态改为"已取消" | 仅"已提交"状态 |
| 删除 | 删除报单记录 | 管理员 |

#### 5. 报单详情弹窗

点击"详情"按钮后显示：

**基本信息**：
- 报单编号
- 快递公司
- 报单日期
- 状态
- 大件/小件/总件数
- 操作员
- 创建时间
- 备注

**包裹明细表**（如果有扫描记录）：
- 序号
- 包裹号
- 类型（大件/小件）
- 扫描时间

## 🔄 状态流转

```
已提交 (submitted)
    ↓
    ├─→ 已确认 (confirmed) ✅ 正常流程
    │
    └─→ 已取消 (cancelled) ❌ 异常情况
```

**状态说明**：
- **已提交**：快递员刚提交的报单，待确认
- **已确认**：管理员确认无误
- **已取消**：发现错误或其他原因取消

## 💡 使用技巧

### 技巧1：快速扫码

1. 准备好扫码枪
2. 打开报单页面
3. 光标定位在"扫描包裹号"输入框
4. 连续扫描，每扫一个选择类型即可
5. 最后点击提交

### 技巧2：批量报单

如果只需要报数量，不需要记录具体包裹号：
1. 直接填写大件/小件数量
2. 不扫描包裹号
3. 点击提交即可

### 技巧3：修正错误

如果发现报单有误：
1. 在Web端找到该报单
2. 点击"取消"按钮
3. 在PDA端重新提交正确的报单

### 技巧4：查看历史

在Web端可以：
1. 按日期范围查询历史报单
2. 按快递公司筛选
3. 查看每个报单的详细包裹列表

## ⚠️ 注意事项

1. **网络要求**：需要连接到公司内网或VPN
2. **权限要求**：需要登录账号才能使用
3. **数据准确性**：提交前请仔细核对数量
4. **包裹号唯一性**：同一报单中不能有重复的包裹号
5. **及时提交**：建议当天的报单当天提交

## 🆘 常见问题

### Q1：扫码后没有反应？

**A**：检查以下几点：
- 扫码枪是否正常工作
- 光标是否在输入框中
- 扫描后是否点击了"添加"按钮

### Q2：提交失败怎么办？

**A**：可能的原因：
- 网络连接问题：检查网络
- 必填项未填：检查快递公司和日期
- 数量为0：至少填写一个数量或添加一个包裹

### Q3：如何修改已提交的报单？

**A**：目前不支持直接修改，需要：
1. 在Web端取消原报单
2. 在PDA端重新提交正确的报单

### Q4：可以删除报单吗？

**A**：可以，但需要管理员权限：
1. 在Web端找到该报单
2. 点击"删除"按钮
3. 确认删除

### Q5：统计数据不准确？

**A**：检查以下几点：
- 是否选择了正确的日期范围
- 是否有筛选条件（快递公司、状态）
- 刷新页面重新加载数据
