# Qt 标签打印客户端 - 使用说明

## 编译要求

1. **Qt 6.5+** (MSVC 版本)
   - 下载: https://www.qt.io/download-qt-installer
   - 安装时选择 MSVC 2019/2022 64-bit 版本
   - 需要勾选 Qt Print Support 组件

2. **Visual Studio 2019 或 2022**
   - 需要安装 "使用 C++ 的桌面开发" 工作负载

3. **CMake 3.16+**
   - 下载: https://cmake.org/download/
   - 或者使用 Visual Studio 自带的 CMake

## 编译步骤

### 方法一：使用编译脚本（推荐）

1. 双击运行 `build.bat`
2. 脚本会自动检测 Qt 路径并编译
3. 编译完成后，可执行文件在 `dist` 文件夹中

### 方法二：使用 Qt Creator

1. 打开 Qt Creator
2. 文件 → 打开文件或项目 → 选择 `CMakeLists.txt`
3. 配置项目（选择 Release 构建）
4. 点击构建按钮

### 方法三：命令行编译

```cmd
# 进入项目目录
cd PrintClientQt

# 创建构建目录
mkdir build
cd build

# 配置（修改 Qt 路径为你的实际路径）
cmake .. -G "Visual Studio 17 2022" -A x64 -DCMAKE_PREFIX_PATH="C:\Qt\6.5.0\msvc2019_64"

# 编译
cmake --build . --config Release

# 部署依赖
C:\Qt\6.5.0\msvc2019_64\bin\windeployqt.exe Release\PrintClient.exe
```

## 部署到其他电脑

编译完成后，`dist` 文件夹包含所有必要的文件：
- `PrintClient.exe` - 主程序
- 各种 Qt DLL 文件
- `platforms` 文件夹
- `styles` 文件夹

将整个 `dist` 文件夹复制到目标电脑即可运行，无需安装 Qt。

## 使用方法

1. **启动程序**
   - 双击 `PrintClient.exe`

2. **启动本地服务**
   - 设置端口（默认 9100）
   - 点击"启动服务"

3. **连接后端服务器**
   - 输入服务器地址（如 `http://192.168.1.100:5000`）
   - 输入客户端名称
   - 点击"注册到服务器"

4. **选择打印机**
   - 从下拉列表选择要使用的打印机
   - 点击"测试打印"验证

## 功能特点

- ✅ 支持任意自定义纸张尺寸（使用 Windows 原生 API）
- ✅ 前端设置什么尺寸，打印什么尺寸
- ✅ 支持文本、条码、二维码、图片、表格等元素
- ✅ 自动心跳保持连接
- ✅ 断线自动重连

## 常见问题

### Q: 编译时找不到 Qt
A: 确保 Qt 安装路径正确，并且安装了 MSVC 版本（不是 MinGW）

### Q: 打印尺寸不对
A: 确保使用最新版本，新版本使用 Windows 原生 API 设置纸张尺寸

### Q: 无法连接到服务器
A: 检查防火墙设置，确保端口 9100 和 5000 已开放

### Q: 前端看不到客户端
A: 
1. 确保客户端显示"已连接到服务器"
2. 在前端点击"刷新"按钮
3. 检查后端控制台是否有注册日志
