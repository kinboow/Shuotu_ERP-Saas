# 标签编辑器功能说明

## 已实现的核心功能 ✓

### 1. 基础标签设计功能

#### 多元元素编辑
- ✓ **文本元素**
  - 自定义字体(宋体、黑体、微软雅黑、楷体、仿宋、Arial等)
  - 字号调整(6-72px)
  - 字体粗细(正常/粗体)
  - 文字颜色选择
  - 多行排版支持
  - 对齐方式(左对齐/居中/右对齐)
  - 旋转角度(-180°到180°)
  
- ✓ **图形元素**
  - 矩形(可设置边框颜色、宽度、填充色)
  - 线条(可设置颜色)
  
- ✓ **条码与二维码**
  - 条码显示(Code128样式)
  - 二维码生成(真实可扫描)
  - 内容可编辑
  
- ✓ **表格元素**
  - 自定义行列数
  - 边框颜色设置
  
- ✓ **图片元素**
  - 图片占位符

#### 精准排版工具
- ✓ **智能对齐辅助线**
  - 画布中心对齐(水平/垂直)
  - 元素间对齐(左/右/上/下/中心)
  - 自动吸附(5px阈值)
  - 实时显示蓝色辅助线
  
- ✓ **元素操作**
  - 拖拽移动
  - 调整大小(右、下、右下三个控制点)
  - 精确位置输入(X/Y坐标)
  - 精确尺寸输入(宽度/高度)
  
- ✓ **图层管理**
  - 置顶
  - 置底
  - 上移一层
  - 下移一层
  
- ✓ **快捷操作**
  - 撤销/重做(Ctrl+Z / Ctrl+Y)
  - 复制/粘贴(Ctrl+C / Ctrl+V)
  - 快速复制(Ctrl+D)
  - 删除(Delete / Backspace)
  - 保存(Ctrl+S)

### 2. 交互体验

#### 编辑模式
- ✓ **单击选中**: 点击元素显示属性面板,可编辑所有属性
- ✓ **拖动移动**: 按住鼠标拖动元素,显示对齐辅助线
- ✓ **双击编辑**: 双击文本类元素直接在画布上编辑内容
- ✓ **调整大小**: 选中元素后拖动控制点调整尺寸

#### 视觉反馈
- ✓ 选中元素显示蓝色虚线边框
- ✓ 选中元素背景半透明高亮
- ✓ 拖动时显示对齐辅助线
- ✓ 操作提示消息

### 3. 工具栏功能

#### 顶部工具栏
- ✓ 返回按钮
- ✓ 撤销/重做
- ✓ 复制/粘贴/快速复制
- ✓ 图层顺序调整(置顶/置底/上移/下移)
- ✓ 删除元素
- ✓ 缩放选择(50%-200%)
- ✓ 保存并返回

#### 左侧工具栏
- ✓ 添加文本
- ✓ 添加条码
- ✓ 添加二维码
- ✓ 添加图片
- ✓ 添加线条
- ✓ 添加矩形
- ✓ 添加表格

#### 右侧属性面板
- ✓ 元素类型显示
- ✓ 内容编辑(文本/条码/二维码)
- ✓ 文本样式(字体/字号/粗细/颜色/对齐/旋转)
- ✓ 位置和尺寸(X/Y/宽/高)
- ✓ 边框设置
- ✓ 图形样式(边框颜色/宽度/填充色)
- ✓ 表格设置(行数/列数)

### 4. 数据管理

- ✓ **历史记录**: 完整的撤销/重做栈
- ✓ **本地存储**: 保存到localStorage
- ✓ **状态传递**: 通过路由state传递设计数据

## 使用技巧

1. **精确定位**: 使用右侧属性面板输入精确的X/Y坐标
2. **快速对齐**: 拖动元素时注意蓝色辅助线,自动吸附对齐
3. **批量操作**: 使用Ctrl+D快速复制多个相同元素
4. **图层管理**: 使用置顶/置底功能调整元素显示顺序
5. **撤销保护**: 所有操作都支持撤销,放心尝试
