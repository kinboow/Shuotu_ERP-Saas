# 快递报单功能 - 安装说明

## 📋 功能概述

快递报单功能允许快递员在手机端（PDA）快速填写报单信息，包括：
- 选择快递公司
- 自动填充当前日期
- 填写大件/小件数量
- 扫码或输入包裹号
- 在电脑端查看所有报单记录

---

## 🗄️ 数据库安装

### 方法1：使用MySQL命令行

```bash
# Windows CMD
type services\database\courier_reports.sql | mysql -u root -p

# 或者登录MySQL后执行
mysql -u root -p
source D:/EER/services/database/courier_reports.sql
```

### 方法2：使用MySQL Workbench

1. 打开MySQL Workbench
2. 连接到数据库
3. 打开文件：`services/database/courier_reports.sql`
4. 点击执行（闪电图标）

### 方法3：手动执行SQL

打开 `services/database/courier_reports.sql` 文件，复制所有SQL语句，在MySQL客户端中执行。

## 📱 使用说明

### 移动端（PDA）- 快递员使用

1. 打开PDA应用：`https://192.168.5.105:3099`
2. 登录后，在首页点击"快递报单"
3. 填写报单信息：
   - 选择快递公司（必填）
   - 报单日期（自动填充今天）
   - 大件数量（可手动输入或通过扫码累加）
   - 小件数量（可手动输入或通过扫码累加）
   - 备注（选填）
4. 扫描包裹号：
   - 在"扫描包裹号"输入框中扫码或手动输入
   - 点击"添加"按钮
   - 选择包裹类型（大件/小件）
   - 包裹会显示在列表中
5. 点击"提交报单"完成

### Web端（电脑）- 管理员查看

1. 打开Web管理端：`https://erp.hlsd.work:3000`
2. 登录后，访问：`/courier-reports`
3. 功能：
   - 查看所有报单记录
   - 按快递公司、状态、日期筛选
   - 查看报单详情（包括所有扫描的包裹）
   - 确认/取消报单
   - 删除报单
   - 查看统计数据（按快递公司汇总）

## 🔧 API接口

### 获取快递公司列表
```
GET /api/courier-companies
```

### 创建报单
```
POST /api/courier-reports
Body: {
  courier_company: "顺丰速运",
  report_date: "2026-01-18",
  large_package_count: 5,
  small_package_count: 10,
  packages: [
    {
      package_no: "SF1234567890",
      package_type: "large",
      scan_time: "2026-01-18T10:30:00"
    }
  ],
  operator_id: "user123",
  operator_name: "张三",
  remark: "备注信息"
}
```

### 获取报单列表
```
GET /api/courier-reports?page=1&pageSize=20&courier_company=顺丰速运&status=submitted
```

### 获取报单详情
```
GET /api/courier-reports/:id
```

### 更新报单状态
```
PUT /api/courier-reports/:id/status
Body: {
  status: "confirmed" // submitted, confirmed, cancelled
}
```

### 删除报单
```
DELETE /api/courier-reports/:id
```

### 获取统计数据
```
GET /api/courier-reports/stats/summary?start_date=2026-01-01&end_date=2026-01-31
```

## 📊 数据库表结构

### courier_companies（快递公司配置表）
- 存储快递公司信息
- 预置了10家常用快递公司

### courier_reports（快递商报单表）
- 存储报单主信息
- 包含大件/小件数量、操作员、状态等

### courier_report_items（快递商报单明细表）
- 存储每个包裹的详细信息
- 包含包裹号、类型、扫描时间等

## 🎯 功能特点

1. **自动填充日期**：打开页面自动填充当前日期
2. **实时统计**：自动计算总件数
3. **扫码支持**：支持扫码枪输入包裹号
4. **类型选择**：每个包裹可选择大件/小件
5. **重复检测**：防止重复添加相同包裹号
6. **列表管理**：可查看和删除已添加的包裹
7. **状态管理**：支持已提交、已确认、已取消三种状态
8. **统计报表**：按快递公司汇总统计数据

## 🔍 故障排查

### 1. API请求失败

检查MISC服务是否正常运行：
```bash
curl -k https://localhost:5000/api/courier-companies
```

### 2. 数据库表不存在

确认已执行数据库初始化脚本：
```sql
USE eer;
SHOW TABLES LIKE 'courier%';
```

### 3. PDA页面无法访问

检查路由配置是否正确：
- `pda-app/src/App.js` 中是否添加了路由
- `pda-app/src/pages/Home.js` 中是否添加了入口

### 4. Web端页面无法访问

检查路由配置是否正确：
- `frontend/src/App.js` 中是否添加了路由和导入
