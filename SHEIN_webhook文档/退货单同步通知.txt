退货单同步通知
/return_order_push_notice
自运营
半托管
该消息用于通知 ERP 平台关于店铺的退货单，具体场景包括客户申请退货和退货单信息更新。消息接入方式可查看文档：webhook说明文档

消息属性
收起
参数名称	数据类型	是否必须	描述
returnOrderNo
string	否	
退货单编号 如：N8DYE00004

changeTime
string	否	
发生时间戳 如：1691140138033

消息示例
收起
{
	  "returnOrderNo": "ND67E08VAR",
	  "changeTime": 1706771656710
}

FAQ
Q：Webhook消息发送失败了，平台会有重试机制吗？
2025-10-30
A：
如果平台将 Webhook 消息推送到开发者的回调地址后，开发者没有及时响应，会触发失败重推机制：非订单消息： 平台只会重试 1次。订单事件： 平台会重试 2次。重试时间： 第一次重试的时间大约为30分钟至1小时之间。如有进一步问题，请联系开放平台的工单支持进行咨询
