## 🚀 部署步骤

### 步骤1：数据库初始化

```bash
# 方法1：使用安装脚本（推荐）
install-courier-report.bat

# 方法2：手动执行SQL
mysql -u root -p < services/database/courier_reports.sql
```

### 步骤2：安装依赖

```bash
cd pda-app
npm install dayjs
```

### 步骤3：重启服务

MISC服务已自动重启（使用nodemon），新功能已生效。

如果手动启动，需要重启：
```bash
cd services/misc
npm run dev
```

### 步骤4：验证功能

```bash
# 运行API测试
test-courier-report-api.bat

# 或手动测试
curl -k https://localhost:5000/api/courier-companies
```
