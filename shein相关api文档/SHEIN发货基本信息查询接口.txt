发货基本信息查询接口
GET
/open-api/shipping/basic
全托管
SHEIN自营
POP
通过该接口获取商家发货地址，发货仓，支持的发货方式，打包方式，车队，自行委托第三方的物流服务商信息；

请求参数（请求头）
收起
参数名称	数据类型	是否必须	描述
Content-Type
string	是	
x-lt-openKeyId
string	是	
访问用户隐私数据时的唯一权限标识 openKeyId
x-lt-timestamp
string	是	
请求时间戳，格式为时间转换为毫秒的值，也就是从1970年1月1日起至今的时间转换为毫秒，如：1583398764000
x-lt-signature
string	是	
接口签名
x-lt-language
string	否	
指定出参语种，默认中文；支持语种:英语:US 法语:FR 西班牙语:ES 德语:DE 中文简体:CN 泰语:TH 巴西葡语:PT 土耳其语：TR
请求参数（查询参数）
收起
参数名称	数据类型	是否必须	描述
orderType
integer	是	
订单类型 1：急采 2：备货
addressId
integer	否	
不同的发货地址可匹配不同的快递物流 不选择发货地址某些物流公司无法匹配
